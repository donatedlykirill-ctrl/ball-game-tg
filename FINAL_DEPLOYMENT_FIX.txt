✅ СНАЧАЛА ПРОВЕРИМ ЛОКАЛЬНО
────────────────────────────────────────────────────────────────

Перед deployment убедись что backend работает локально:

1. Откой PowerShell

2. Перейди в папку backend:
   cd "C:\Users\Пользователь\Desktop\home\BALL GAME TG\backend"

3. Установи зависимости:
   npm install

4. Запусти сервер:
   npm start

5. Должно вывести:
   "Ball Game Backend running on PORT 3000"

6. Открой в браузере:
   http://localhost:3000/health

7. Должно вернуть:
   {"status":"ok"}

8. Если всё хорошо - Ctrl+C чтобы остановить
