GITHUB UPLOAD INSTRUCTIONS
===========================

STEP 1: Create GitHub Repository
1. Go to https://github.com/new
2. Repository name: ball-game-tg
3. Description: Ball Arena Game with Backend
4. Visibility: Public
5. Add .gitignore: Node
6. Click "Create repository"

STEP 2: After creation, copy the URL
Example: https://github.com/YOUR_USERNAME/ball-game-tg.git

STEP 3: Run these commands (replace YOUR_USERNAME with your GitHub username)

git remote add origin https://github.com/YOUR_USERNAME/ball-game-tg.git
git add .
git commit -m "Initial commit - Ball Arena with MySQL Backend"
git push -u origin main

STEP 4: When prompted for password
- Go to https://github.com/settings/tokens
- New token (classic)
- Select: repo scope
- Generate token
- Copy and paste it as password
- Press Enter

STEP 5: Done!
Your code is now on GitHub. Ready for Vercel deployment!

Next: Follow FINAL_DEPLOYMENT.md for Vercel setup
